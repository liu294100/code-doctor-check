# 视频生成 Skills

基于浏览器自动化和 HTML 渲染的视频生成 Kiro Skills 集合。

## Skills 列表

| Skill | 目录 | 说明 |
|-------|------|------|
| HTML2Video | `html2video/` | HTML 模板驱动的视频生成 |
| Browser-Use-Video | `browser-use-video/` | 浏览器自动化视频生成 |

## 使用方式

在 Kiro 聊天中使用 `#skill-name` 引用对应的 Skill。例如：

```
#html2video 帮我生成一个产品演示视频
```
