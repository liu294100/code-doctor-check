# Python SDK Skills

Python SDK 开发相关的 Kiro Skills 集合。

## Skills 列表

| Skill | 文件 | 说明 |
|-------|------|------|
| SDK 开发规范 | `python-sdk-dev.md` | Python SDK 开发规范和最佳实践 |
| SDK 代码审查 | `python-sdk-review.md` | Python SDK 代码质量审查 |
| SDK 打包发布 | `python-sdk-package.md` | PyPI 打包发布流程指导 |

## 使用方式

在 Kiro 聊天中使用 `#skill-name` 引用对应的 Skill。例如：

```
#python-sdk-dev 帮我开发一个 SDK
```
