# Flask Web 应用 Skills

Flask Web 应用开发相关的 Kiro Skills 集合。

## Skills 列表

| Skill | 文件 | 说明 |
|-------|------|------|
| Flask 开发指导 | `flask-dev.md` | Flask Web 应用开发指导和最佳实践 |
| Flask 安全审查 | `flask-review.md` | Flask 应用安全审查 |
| Flask Docker 部署 | `flask-package.md` | Flask 应用 Docker 容器化部署 |

## 使用方式

在 Kiro 聊天中使用 `#skill-name` 引用对应的 Skill。例如：

```
#flask-dev 帮我开发一个 Flask 应用
```
