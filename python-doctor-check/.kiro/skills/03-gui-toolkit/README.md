# GUI 桌面工具 Skills

Python GUI 桌面小工具开发相关的 Kiro Skills 集合。

## Skills 列表

| Skill | 文件 | 说明 |
|-------|------|------|
| GUI 开发指导 | `gui-dev.md` | GUI 框架选型与开发指导 |
| GUI 跨平台审查 | `gui-review.md` | GUI 应用跨平台兼容性审查 |
| GUI 打包发布 | `gui-package.md` | PyInstaller 打包发布流程 |

## 使用方式

在 Kiro 聊天中使用 `#skill-name` 引用对应的 Skill。例如：

```
#gui-dev 帮我开发一个桌面小工具
```
