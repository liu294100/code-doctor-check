# 文档处理 Skills

面向文档阅读、设计、写作的 Kiro Skills 集合。

## Skills 列表

### 文档阅读器 (docs-readers)

| Skill | 目录 | 说明 |
|-------|------|------|
| PPT 阅读 | `docs-readers/read-ppt/` | PowerPoint 文档内容提取与分析 |
| PDF 阅读 | `docs-readers/read-pdf/` | PDF 文档内容提取与分析 |
| 图片阅读 | `docs-readers/read-image/` | 图片内容识别与文字提取 |
| Excel 阅读 | `docs-readers/read-excel/` | Excel 表格数据提取与分析 |
| Word 阅读 | `docs-readers/read-docx/` | Word 文档内容提取与分析 |
| JAR 反编译 | `docs-readers/jar-decompile/` | Java JAR 文件反编译分析 |
| APK/JADX 反编译 | `docs-readers/jadx-decompile/` | Android APK 反编译分析 |

### 设计工具

| Skill | 目录 | 说明 |
|-------|------|------|
| UI 图表设计 | `ui-diagram-designer/` | UI 流程图、架构图设计 |
| DrawIO 读取 | `read-drawio/` | DrawIO 文件内容读取与分析 |
| DrawIO 设计 | `drawio-designer/` | DrawIO 图表设计辅助 |
| 文档写作 | `doc-writer/` | 技术文档写作辅助 |

## 使用方式

在 Kiro 聊天中使用 `#skill-name` 引用对应的 Skill。例如：

```
#read-pdf 帮我分析这个 PDF 文档
```
