# 文档阅读器 Skills

多格式文档内容提取与分析的 Kiro Skills 集合。

## Skills 列表

| Skill | 文件 | 说明 |
|-------|------|------|
| PPT 阅读 | `read-ppt/SKILL.md` | PowerPoint 文档内容提取与分析 |
| PDF 阅读 | `read-pdf/SKILL.md` | PDF 文档内容提取与分析 |
| 图片阅读 | `read-image/SKILL.md` | 图片内容识别与文字提取 |
| Excel 阅读 | `read-excel/SKILL.md` | Excel 表格数据提取与分析 |
| Word 阅读 | `read-docx/SKILL.md` | Word 文档内容提取与分析 |
| JAR 反编译 | `jar-decompile/SKILL.md` | Java JAR 文件反编译分析 |
| APK/JADX 反编译 | `jadx-decompile/SKILL.md` | Android APK 反编译分析 |

## 使用方式

在 Kiro 聊天中使用 `#skill-name` 引用对应的 Skill。例如：

```
#read-pdf 帮我分析这个 PDF 文档
```
