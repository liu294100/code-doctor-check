# 代码质量审查 Skills

Java 技术栈代码质量审查相关的 Kiro Skills 集合。

## Skills 列表

| Skill | 文件 | 说明 |
|-------|------|------|
| Java 代码审查 | `java-code-review.md` | Java 代码规范、质量、最佳实践审查 |
| SQL 脚本审查 | `sql-script-review.md` | SQL 脚本规范、性能、安全审查 |
| Spring Boot SQL 审查 | `springboot-sql-review.md` | Spring Boot 项目中 SQL 使用审查 |
| MR 代码审查 | `merge-review.md` | GitLab Merge Request 代码审查 |
| GitLab MR 报告 | `gitlab-mr-report.md` | 生成 GitLab MR 审查报告 |

## 使用方式

在 Kiro 聊天中使用 `#skill-name` 引用对应的 Skill。例如：

```
#java-code-review 帮我审查这段代码
```
