# 券商业务系统 Skills

券商核心业务系统专项 Kiro Skills 集合。

## Skills 列表

| Skill | 文件 | 说明 |
|-------|------|------|
| 交易系统 | `brokerage-trading.md` | 港股/美股/A股交易系统业务审查 |
| 资金系统 | `brokerage-capital.md` | 保证金/融资/清算资金系统审查 |

## 使用方式

在 Kiro 聊天中使用 `#skill-name` 引用对应的 Skill。例如：

```
#brokerage-trading 帮我审查交易系统设计
```
